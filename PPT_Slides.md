## [슬라이드 3] 연구 배경

**고막 천공 (Tympanic Membrane Perforation)**

- 고막에 구멍이 생기는 질환
- 원인: 중이염, 외상, 압력 변화
- 결과: 전음성 난청 발생

**Air-Bone Gap (ABG)**
- 기도 청력 - 골도 청력
- 높을수록 전음성 난청 심함
- 단위: dB

---

## [슬라이드 4] 연구 목적

**목표**
- 고막 천공이 청력에 미치는 영향을 물리 기반으로 모델링
- 천공 크기, 위치, 주파수에 따른 ABG 예측
- 실제 환자 데이터와 비교 검증

**의의**
- 수술 전 청력 손실 예측 도구
- 환자 교육 및 시각화

## [슬라이드 5] 시뮬레이션 모델 구조

**최종 ABG 계산식**

```
ABG = [물리 모델] × 50% + [문헌 데이터] × 50%
```

**물리 모델 구성요소**
1. 크기 효과 (Size Effect)
2. 위치 효과 (Position Effect)
3. 볼륨 효과 (Volume Effect)
4. 주파수 변조 (Frequency Modulation)
5. Round Window Shielding Effect

---

## [슬라이드 6] 물리 모델 - 크기 효과

**수식**
```
ABG_size = 20 × log₁₀(1 / (1 - 천공비율))
```

**원리**
- 천공이 클수록 ABG 증가
- 비선형 관계 (로그)
- 음향 임피던스 변화 기반

**천공 등급**
| Grade | 비율 | 직경 |
|-------|------|------|
| I | <25% | ~2.5mm |
| II | 25-50% | ~4.0mm |
| III | 50-75% | ~5.5mm |
| IV | >75% | ~7.0mm |

---

## [슬라이드 7] 물리 모델 - 위치 효과

**위치에 따른 ABG 변화**

| 위치 | 효과 | 이유 |
|------|------|------|
| 후방 | +5 dB | 이소골 근처 |
| 전방 | -4 dB | 이소골에서 멂 |
| 상방 | +3 dB | 추골 연결점 |
| 후상방 | +4 dB 추가 | 최대 손실 |

**총 범위: -8 ~ +12 dB**

참고: Mehta et al. 2006

---

## [슬라이드 8] 물리 모델 - 주파수 효과

**주파수별 변조 계수**

| 주파수 | 계수 |
|--------|------|
| < 500 Hz | ×1.3 |
| 500-1000 Hz | ×1.15 |
| 1000-2000 Hz | ×1.0 (기준) |
| > 2000 Hz | ×0.85 |

**특징**
- 저주파에서 천공 영향 큼
- 고주파에서 영향 감소

---

## [슬라이드 9] Round Window Shielding Effect

**현상**
- Grade IV (>75% 천공)에서 ABG가 오히려 감소
- Grade III보다 Grade IV의 청력이 더 좋음

**원리**
- 큰 천공 → 소리가 정원창에 직접 도달
- 이소골 직접 자극 가능
- 최대 20 dB 감소 효과

**근거**
- Voss et al. 2001
- Merchant et al. 1997

---

## [슬라이드 10] 환자 데이터 구성

**데이터셋**
- ICW (Intact Canal Wall): 30명
  - 고막 천공만 있음
  - 이소골 정상
- CWD (Canal Wall Down): 63명
  - 진주종 동반
  - 이소골 부식

**Grade별 환자 수**
| Grade | 환자 수 |
|-------|---------|
| I | 1명 |
| II | 4명 |
| III | 1명 |
| IV | 2명 |

## [슬라이드 11] 데이터 한계

**문제점**
1. Grade별 환자 수 부족 (통계적 한계)
2. 순수 천공 환자 드묾 (대부분 SNHL 동반)
3. Grade III 환자(ICW4): 혼합성 난청

**골도 청력 확인**
- 정상: ≤ 20 dB
- ICW4: 55-60 dB (비정상)
- 순수 천공 비교 어려움

## [슬라이드 17] 주요 발견 요약

**1. Round Window Shielding**
- Grade IV에서 ABG 감소 현상 재현

**2. 위치의 중요성**
- 같은 크기라도 위치에 따라 최대 10 dB 차이
- 후상방 천공이 가장 심함

**3. 모델 유효성**
- 대부분 ±5 dB 이내 오차
- 물리 기반 예측 가능

## [슬라이드 18] 모델 한계

1. **이소골 부식 미반영**
   - CWD 환자와 차이 존재

2. **순수 천공 데이터 부족**
   - 대부분 복합 질환

3. **개인차 미반영**
   - 중이강 볼륨, 고막 두께 변이

4. **동적 효과 미반영**
   - 시간에 따른 치유/악화

## [슬라이드 19] 결론

**시뮬레이션 유효성**
- 물리 기반 모델로 ABG 예측 성공
- 임상 데이터와 ±5 dB 이내 일치

**임상적 의의**
- 수술 전 청력 손실 예측 도구
- 천공 위치 기반 수술 우선순위 결정
- 환자 교육 및 시각화

**향후 연구**
- 이소골 부식 모델 추가
- 더 많은 순수 천공 데이터 수집
